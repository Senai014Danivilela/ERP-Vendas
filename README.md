# ERP-Vendas

## Descrição

Este projeto foi desenvolvido durante o estágio na empresa **Domtec Sistemas**, como parte do programa de formação profissional. O sistema foi construído seguindo os aprendizados adquiridos no curso de **Delphi com SQL Server** fornecido pela Udemy Academy, disponibilizado pela própria empresa.

## Objetivo

O ERP-Vendas é um sistema de gestão de vendas desenvolvido em Delphi, com banco de dados SQL Server, visando a automatização e controle de processos comerciais e operacionais.

## Tecnologias Utilizadas

- **Linguagem**: Delphi
- **Banco de Dados**: SQL Server
- **Formação**: Udemy Academy (Curso de Delphi com SQL Server)

## Estrutura do Projeto

O projeto está organizado para facilitar a manutenção e evolução do sistema de vendas, com módulos específicos para gestão de operações comerciais.
